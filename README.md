Este repositorio contiene los archivos necesarios para la implementación de un sistema de monitoreo ambiental basado en IoT. El archivo principal en MicroPython corresponde al código ejecutado en el microcontrolador ESP32, el cual se encarga de la adquisición de datos de temperatura y humedad mediante el sensor DHT22, así como del envío de esta información hacia un servidor local. Por otro lado, el archivo insertar.php funciona como intermediario en el servidor Apache, permitiendo recibir y almacenar los datos en una base de datos MySQL. El script de base de datos define la estructura necesaria para el almacenamiento de las variables ambientales y su registro temporal. Finalmente, el archivo index.php corresponde a la interfaz web, en la cual se visualizan los datos recolectados a través de tablas y gráficas dinámicas, permitiendo al usuario monitorear en tiempo real las condiciones del entorno.
